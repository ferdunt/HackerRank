# Funny String - Easy

The solution to this problem is written in JavaScript the detail about the problem is below and you can check the solution above.

## 🥚 Input 🥚

```
"lmnop" (String)
```

## 🐣 Output 🐣

```
"Funny" (String)
```

## 👓 Explanation 👓

```
1# "lmnop"
2# ["lmnop", "ponml"]
3# [[108,109,110,111,112], [112,111,110,109,108]]
4# [[1,1,1,1],[1,1,1,1]]
5# "1111" === "1111" (True)
6# "Funny" 🎉
```
