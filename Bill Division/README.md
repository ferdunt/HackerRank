# Bill Division - Easy

The solution to this problem is written in JavaScript the detail about the problem is below and you can check the solution above.

## 🥚 Input 🥚

```
([3, 10, 2, 9], 1, 12) (Array of Numbers, Number, Number)
```

## 🐣 Output 🐣

```
5 (Number)
```

## 👓 Explanation 👓

```
1# ([3, 10, 2, 9], 1, 12)
2# ([3, 0, 2, 9], 12)
3# ([3, 2, 9], 12)
4# ([14], 12)
5# ([14/2], 12)
6# ([7], 12)
7# (12 - 7)
8# 5 🎉
```
