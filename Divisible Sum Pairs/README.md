# Divisible Sum Pairs - Easy

The solution to this problem is written in JavaScript the detail about the problem is below and you can check the solution above.

## 🥚 Input 🥚

```
n = 6 (Number)
ar = [1, 3, 2, 6, 1, 2] (Array of Numbers)
k = 3 (Number)
```

## 🐣 Output 🐣

```
5 (Number)
```

## 👓 Explanation 👓

```
1# (i,j)
2# (0,2)
3# (0 < 2) => true
4# (ar[0],ar[2])
5# (1,2)
6# (1 + 2)
7# 3 % k === 0
8# 3 % 3 === 0 => true
9# 1 🎉
```
