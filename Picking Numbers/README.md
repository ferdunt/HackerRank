# Picking Numbers - Easy

The solution to this problem is written in JavaScript the detail about the problem is below and you can check the solution above.

## 🥚 Input 🥚

```
[1, 1, 2, 2, 4, 4, 5, 5, 5] (Array of Numbers)

```

## 🐣 Output 🐣

```
5 (Number)
```

## 👓 Explanation 👓

```
1# [1, 1, 2, 2, 4, 4, 5, 5, 5]
2# [1, 1, 2, 2], [4, 4, 5, 5, 5]
3# [4], [5]
4# 5 🎉
```
